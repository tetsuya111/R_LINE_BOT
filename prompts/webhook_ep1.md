# プロンプトのログ
`logs/`以下

# 要件
- LINEのMessaging APIでwebhookのエンドポイントを作成すること

# 基本設計
- メッセージを受信するエンドポイントを作成すること
- エンドポイントはfastapiで作成すること
- エンドポイントはngrokでweb上に公開すること
- LINEユーザーからメッセージを受信した場合
- 以下のパラムを指定できること
  - メッセージ
  - 宛先ユーザー一覧
    - リスト型
    - 宛先ユーザーはユーザー名で取得し、プログラム上でユーザーIDを特定すること
- pythonで作成すること
- APIキーなどは.envファイルで設定できること
